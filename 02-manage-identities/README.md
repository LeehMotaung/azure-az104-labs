# Create, Configure, and Manage Identities

## Overview
This lab focused on Microsoft Entra ID identity management.

## Tasks Completed
- Created users
- Restored deleted users
- Created groups
- Assigned licenses
- Configured device registration
- Created custom security attributes

## Key Takeaways
- Microsoft Entra ID centralizes identity management.
- Group-based licensing simplifies administration.
- Device registration improves security.
